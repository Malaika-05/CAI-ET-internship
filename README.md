# Computer Vision & Intelligent Document Understanding — Internship

## 📌 Repository Description

A hands-on internship program focused on building a strong foundation in **Computer Vision**, **Image Processing**, and **OCR (Optical Character Recognition)** for intelligent document understanding. This repo contains weekly learning materials, mini-projects, sample datasets/images, and code implementations covering the journey from basic pixel manipulation to advanced document AI pipelines.

---

## 🎯 Program Objective

To build practical, project-driven skills in:

- Digital image representation and processing
- Computer vision fundamentals using OpenCV
- Optical Character Recognition (OCR) for document understanding
- Real-world intelligent document processing pipelines

Each week builds on the previous one — starting with core image processing and progressing toward applied CV/OCR systems.

---

## 🧰 Tech Stack

- **Language:** Python
- **Core Libraries:** OpenCV, NumPy, Pillow, Matplotlib, scikit-image
- **OCR (upcoming):** Tesseract / EasyOCR (TBD based on later weeks)
- **Environment:** VS Code / Jupyter Notebook, virtual environment (venv)

## ✅ Completed Weeks

### Week 1–2: Python, Computer Vision & Image Processing

**Learning Outcome:** Load and preprocess images using OpenCV.

Topics covered:

- Python environment setup & core CV libraries installation
- Digital image representation (pixels, RGB, grayscale)
